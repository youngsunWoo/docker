# 1.도커

# 1.1 도커란?

리눅스(LXC) 컨테이너 기반의 오픈소스 가상화 플랫폼 Immutable Infrastructure
서버에 깔린 OS와 서비스 운영을 위한 환경을 분리해서, 서비스 운영환경을 패키징하여 배포
